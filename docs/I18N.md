# 多语言支持

AutoDraw 现在支持以下三种语言：

## 支持的语言

- 🇨🇳 **简体中文** (zh-CN)
- 🇺🇸 **英文** (English)
- 🇰🇷 **韩文** (한국어)

## 使用方法

### 切换语言

在应用顶部工具栏的右侧，你会看到一个语言选择下拉框。点击即可切换语言：

1. 点击语言选择器
2. 选择你想要的语言
3. 界面会立即切换到所选语言

### 默认语言

应用会自动检测你的浏览器语言设置：

- 如果浏览器语言是中文，默认显示简体中文
- 如果浏览器语言是韩文，默认显示韩文
- 其他情况默认显示英文

### 语言持久化

你选择的语言会自动保存到浏览器的本地存储中，下次打开应用时会自动使用你上次选择的语言。

## 技术实现

### 使用的技术栈

- **vue-i18n**: Vue 3 官方国际化库
- **localStorage**: 保存用户语言偏好

### 文件结构

```
src/
├── locales/
│   ├── index.ts        # i18n 配置
│   ├── zh-CN.ts        # 简体中文翻译
│   ├── en.ts           # 英文翻译
│   └── ko.ts           # 韩文翻译
├── composables/
│   └── useAnchorOptions.ts  # 锚点选项的动态翻译
└── main.ts             # 注册 i18n 插件
```

### 添加新翻译

如果你想添加更多语言或修改现有翻译：

1. 编辑对应语言文件（如 `src/locales/zh-CN.ts`）
2. 保持所有语言文件的键名一致
3. 使用 `t('key.path')` 在组件中引用翻译文本

### 示例

```vue
<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
</script>

<template>
  <h1>{{ t('app.title') }}</h1>
  <button>{{ t('header.uploadExcel') }}</button>
</template>
```

## 翻译覆盖范围

✅ 所有界面文本已翻译：
- 顶部工具栏
- 字段列表
- 字段编辑器
- 画布设置
- 预览区域
- 按钮和标签
- 锚点选项
- 提示信息

## 注意事项

- 占位符（如 `{字段名}`）的格式在所有语言中保持一致
- Excel 列名匹配不受语言影响
- 配置文件导入导出与语言无关
